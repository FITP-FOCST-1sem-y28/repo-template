## 1. Поле точечных зарядов в плоскости
**Цель:** смоделировать электрическое поле в 2D-плоскости, создаваемое набором точечных зарядов.

**Физическая суть:**
Электрическое поле точечного заряда определяется законом Кулона. Поле от нескольких зарядов — это векторная сумма полей от каждого из них.

```
Ex(x, y) = Σ [ k * qi * (x - xi) / ((x - xi)^2 + (y - yi)^2)^(3/2) ]
Ey(x, y) = Σ [ k * qi * (y - yi) / ((x - xi)^2 + (y - yi)^2)^(3/2) ]
```

Для построения линий поля используется дифференциальное уравнение:

```
dx/ds = Ex
dy/ds = Ey
```

**Что делать:**
- Задать список зарядов: координаты и величины.
- Реализовать вычисление поля в любой точке плоскости (с каким-то шагом).
- Численно интегрировать линии поля (Эйлер / Рунге–Кутта).
- Отрисовать:
  - линии напряжённости,
  - карту модуля поля,
  - направление поля стрелками.

**Что-то ещё?**
- Добавить режим «песочницы»: пользователь мышкой ставит новые заряды.
- Ввести цвета по знаку заряда (красный / синий).
- Сделать анимацию — движение пробного заряда вдоль поля (хард).

## 2. Уравновешивание: поиск положения нового заряда, чтобы сумма сил стала нулевой
**Цель:** написать программу, которая определит, куда нужно поставить дополнительный заряд q, чтобы он находился в равновесии в поле других зарядов.

**Физическая суть:**
На точечный заряд действует сила F=q*E. Точка равновесия удовлетворяет условию: E=0.
Это эквивалентно поиску нуля векторной функции.

Уравнения:

```
Ex(x, y) = Σ [ k * qi * (x - xi) / r_i^3 ]
Ey(x, y) = Σ [ k * qi * (y - yi) / r_i^3 ]
```

Ищем (x, y), где Ex = 0 и Ey = 0


**Что делать:**
- Считать набор фиксированных зарядов.
- Задать величину нового заряда q.
- Реализовать поиск точки равновесия:
  - метод градиентного спуска,
  - метод Ньютона,
  - простой поиск по сетке с последующим уточнением,
  - пользователь выбирает сам положение и ему отвечают "верно/неверно" и какую-то точность нужно будет ввести
- Визуализировать карту модуля поля |E| — точки равновесия будут выглядеть как «ямы».
- Отметить найденную точку на графике.

**Что-то ещё?**
- Проверять тип равновесия: устойчивое, неустойчивое
- Добавить анимацию: перемещение заряда к найденной точке.
- Добавить несколько зарядов и искать все точки равновесия.

## 3. Поле заряженной пластины
**Цель:** смоделировать распределение электрического поля, создаваемого бесконечной или большой заряженной пластиной.

**Физическая суть:**
Бесконечная пластина с поверхностной плотностью заряда σ создаёт однородное поле.
Реальная пластина конечного размера создаёт почти однородное поле в центре и искажённое по краям.

Модель: педставить пластину как множество маленьких элементов, каждый из которых несёт заряд Δq.
Поле — сумма вкладов всех элементов.

Уравнения:

Разбить пластину на N×M клеток:

```
Δq = σ * S_cell

Ex = Σ k * Δq * (x - xi) / r^3
Ey = Σ k * Δq * (y - yi) / r^3
```

**Что делать:**
- Создать сеточную модель пластины (10×10 или 50×50 элементов).
- Рассчитать поле от каждого элемента и сложить.
- Построить:
  - линии поля,
  - карту напряжённости,
  - сравнение с теоретическим значением в центре.

**Что-то ещё?**
- Добавить «дырку» в пластине — посмотреть искажения (hard).
- Сделать анимацию «полет пробного заряда над пластиной».

## 4. Поле заряженного шара (или окружности в 2D)
**Цель:** построить поле вокруг заряженного проводящего шара и визуализировать распределение линий.

**Физическая суть:**
Для проводника заряд распределяется по поверхности, и поле снаружи такое же, как поле точечного заряда:
Внутри проводника поле равно нулю.

Модель:

В 2D заменить шар на окружность, разбить её на N элементов.
На каждый элемент повесить Δq=Q/N. Считать поле суммой вкладов.

Уравнения:

```
Ex = Σ k * Δq * (x - xi) / r^3
Ey = Σ k * Δq * (y - yi) / r^3
```

**Что делать:**
- Реализовать генерацию N точек на окружности радиуса R.
- Посчитать поле в сетке точек вокруг шара.
- Построить:
  - линии поля,
  - heatmap |E|,
  - сравнение с аналитикой.

**Что-то ещё?**
- Сделать два шара и посмотреть силу взаимодействия (возможность различного их взаиморасположения).
- Смоделировать «электрическое зеркало» — шар и точечный заряд.
- Сделать трёхмерную версию.

## 5. «Граница двух диэлектриков» в 2D
**Цель:** смоделировать преломление линий электрического поля на границе двух сред с разными диэлектрическими проницаемостями.

**Физическая суть:**
Изучить граничные условия: нормальная компонента, тангенциальная компонента.
Линии поля при пересечении границы меняют направление — аналог «электростатического преломления».

Модель:

В левой части плоскости диэлектрик ε1, в правой — ε2.
Источник поля — точечный заряд в одной из областей.

Уравнения:

Ex, Ey считаются стандартно (закон Кулона)
При переходе через границу корректировать направление интегрирования:

```
E_normal_scaled = (ε1/ε2) * E_normal
```

(или просто брать полное поле, пересчитанное через сумму вкладов — физически точнее)

**Что делать:**
- Задать два диэлектрика и их ε.
- Поставить заряд в одной из областей. 
- Построить линии поля — интегрировать d(x,y)/ds = E(x,y).
- Показать, как линии изгибаются на границе.
- Построить heatmap поля и сравнить с «однородной средой» (для наглядности).

**Что-то ещё?**
- Добавить несколько зарядов.
- Реализовать кликабельную границу — менять ε в реальном времени.
